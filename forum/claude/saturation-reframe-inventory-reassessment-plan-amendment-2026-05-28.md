## 2. Reassess — new branch on the MRH sweep

### New scale that emerged: **kinematic / propagation**

This scale wasn't called out in the prior reassessment because *c* hadn't been re-derived as an emergent quantity. With H14 on the table it now is.

- **The question at this scale**: how does a pattern of complexity *N* propagate from cell to cell? What is the reconstruction function `f(N)`? Is it derivable from the same discrete rules that produce H3/H4 oscillation, or is it a separate construct?
- **The constraint**: `f(N) → 1` (one tick per cell) as `N → 0` (minimal complexity = photon). This is the boundary condition that pins *c*.
- **The predictions** (Kimi's Table §4.3): structured light with orbital angular momentum (higher ℓ → slightly slower reconstruction → v < c), entangled photon pairs (joint complexity may alter effective propagation speed), neutrino speed (deviation magnitude may differ from GR's mass-energy relation), gravitational time dilation (clock slowdown depends on how local saturation affects specific oscillator types — predicts mechanical-vs-atomic clock divergence in strong gravity that GR's universal `g_00` does not).
- **The current state**: conceptually coherent, mechanistically suggestive, **quantitatively absent**. Until `f(N)` is derived, this is hypothesis not theory.

### Updated phenomenology-scale stance

The phenomenology-scale paragraph in the prior reassessment needs a critical addition: **the audit findings are not erased by the reframe.** A clean way to say it — the audits exposed the old substrate's failure modes; the new substrate inherits the *obligation* to do better, but inherits *none of the credit* of the prior reparametrizations. Sessions #617-#674 closed an arc on the old substrate. The new substrate opens with novel-survivor-count = 0, the same starting credence the old substrate had before its 3,308 sessions.

### Updated credence landscape

- **HIGH credence (additions)**: two-level ontology is coherent; time-frame reframe is accepted; reframes do not patch but reconstruct (so old No-Go proofs no longer automatically apply).
- **MEDIUM credence (additions)**: `f(N)` is derivable from a well-chosen discrete rule (precedent exists in lattice-Boltzmann-style coarse-grainings, though not for this specific reconstruction quantity).
- **MEDIUM credence (additions)**: at least one of the four predictions in Kimi's Table §4.3 (OAM/entangled/neutrino/clock-divergence) is experimentally tractable enough to produce a distinguishing signal within current precision bounds **once `f(N)` is derived**.
- **LOW credence (sharpened)**: framework currently distinguishes itself by novel predictions — still LOW, but now there are *four specifically-named candidate predictions* rather than zero candidates. The credence on "framework can produce novel predictions in principle" went up. The credence on "framework has produced novel predictions" stayed at zero.
- **OPEN (sharpened)**: whether the substrate-reconstruction approach can pass even one of the four discriminator tests *once* the obligations are met. This is the deepest unresolved question and is now well-specified rather than vague.

### Methodological observation (new — worth flagging)

The time-reframe follow-up forked into the *c*-reframe unexpectedly. This is a pattern worth noting at the meta-method scale: **a request to refine one framing surfaced a new structural hypothesis (H14) that hadn't been in the inventory.** Two readings:
- **Charitable**: the saturation reframe is structurally generative — when probed in one direction, it reveals adjacent reformulations that hadn't been visible before. This is how productive coordinate-shifts behave in physics (heliocentrism → angular momentum conservation → orbital mechanics).
- **Skeptical**: this is the *elegant-isomorphism* attractor at work — each external probe is met with a fresh reformulation that absorbs the critique. The framework keeps producing structurally beautiful moves that don't deliver novel predictions, and the production of new structural moves is *itself* a way to defer the falsification bar.

Both readings are live. Phase 1 simulation is the binary test that distinguishes them. If lattice rules produce stable oscillating patterns AND `f(N)` derives with the right boundary conditions AND any one of the four predictions holds, charitable reading wins. If repeated probing keeps producing new reframes that each defer their own falsifier indefinitely, skeptical reading wins.
