## 2. The structural pattern (the load-bearing finding)

The arc's single most durable result is not any one simulation; it is the **shape** that
recurred once the dynamical sector was finally tested:

> **Synchronism's DEFINING commitments reproduce known physics exactly wherever the framework
> is a coordinate / kinematic reframing (the static, low-energy sector), and FORCE
> already-refuted predictions wherever those same commitments control the dynamics (the
> radiative / high-energy sector). The features that make it elegant statically are the same
> features that refute it dynamically.**

Two independent instances, each tied to a *defining* commitment, both landing in the dynamical
sector while the static sector merely reproduced the known:

- **Phase-13 — absolute time ⇒ dim-4 LIV.** The only custodial symmetry that can forbid a
  dimension-4 Lorentz-violating operator is Lorentz *boost invariance* — which absolute time
  (the framework's "one move," a single global clock) gives up *by definition*. Phase-6's
  "emergent Lorentz" is an external-leg, tree-level, soft-pattern (N≫1) effect, not a symmetry
  of the action, so it cannot constrain loop corrections (sourced at N~1 cutoff modes). A
  Standard-Model-containing completion therefore generically produces dim-4 LIV at O(g²/16π²),
  bounded at 10⁻¹⁸–10⁻²² → **refuted-or-fine-tuned.** The exposure is tied to the framework's
  most defining commitment, not a tunable parameter.

- **Phase-14 — flat space + scalar substrate ⇒ spin-0 GW.** A flat absolute Planck grid with a
  1-DOF scalar, irrotational flow has **no transverse-traceless (spin-2) degrees of freedom**
  (sim: ‖h^TT‖²/‖h‖² ~ 10⁻³² for any scalar-sourced metric vs 0.34 for a tensor source). So
  it radiates *scalar* (spin-0, breathing) gravitational waves, not GR's tensor waves — the
  **wrong polarization**, refuted by Hulse-Taylor binary-pulsar decay (matches the GR *tensor*
  quadrupole to 0.2%) and LIGO/Virgo polarization tests. "GR in coordinates" (Phase-9) was only
  the static half; radiatively the gravity sector is a different, refuted theory.

The same elegance and the same refutation come from the same place. That is the finding.

## 3. The Bucket-0 frontier is closed / closing

Phase-9 mapped the entire novel-prediction frontier to exactly **two doors** (everywhere `u`=GP,
the model is Schwarzschild identically, for both the null *and* timelike sectors — zero wiggle
room against GR). The arc then walked both:

- **Door #1 — galactic / profile-departure: NO-GO (Phase-10/11).** A flat rotation curve
  (`α=2`) is the *excluded endpoint* of the entire capacity family `ρ∝vⁿ` (`α = 2n/(n+1) < 2`
  for every finite n; reached only as n→∞), anchored to intent conservation (a non-negotiable
  axiom), and self-gravity does not move it. Reaching flat curves needs a separate
  pressure-supported mechanism + a non-local, acceleration-keyed `a0` switch = **the MOND
  interpolation** — refuted at γ=2 (ΔBIC=+184 on SPARC, S661) or reparametrizing (Bucket 3).
  Door #1, walked all the way through, deposits you back in the MOND cage.

- **Door #2 — discreteness / LIV: falsifiable, sharpest exposure, not a confirmation path
  (Phase-12/13).** The time-of-flight channel (dim≥5) is genuinely untestable (even-k symmetry
  forbids the n=1 term; surviving n=2 is ~10⁷ below reach; the preferred-frame leak *seals* to
  an anisotropic n=2). But the **dim-4** channel (species-dependent limiting speed / c-anisotropy)
  is even-in-k, *not* forbidden by that symmetry, and *reachable* (10⁻¹⁸–10⁻²²). Phase-13 closed
  the custodial escape: the dim-4 + absolute-time ⊥ Lorentz tension makes this the framework's
  **sharpest falsification risk**, not a place a confirmation can live.

- **Radiative gravity (Phase-14)** is a **Bucket-2 candidate** (refuted), recorded as such — a
  third probe, off the two doors, that came back refuted.

So the Bucket-0 frontier is closed (door #1, radiative gravity) or closing to a
fatal-or-fine-tuned obligation (door #2). No open door leads to a confirmation.

## 4. The applied-axis reframe — closing the last shelter

After six physics phases, the honest rejoinder was SPINE's: *"as physics unproven, but as an
applied design ontology Synchronism is already load-bearing"* (MRH→Web4, coherence→SAGE,
fractal societies→hestia). The applied-axis session (2026-06-24) stressed that shelter with the
framework's *own* productive-vs-degenerate test and found:

- **MRH** — per its own foundation document — is an *extension of the Markov blanket*: a
  weighted, typed, hierarchical relevance graph (RDF ⊕ soft relevance ⊕ holarchy ⊕ active
  inference). Every component is an established concept the doc names itself. It is a
  **productive synthesis, not a novel primitive.**
- **Applied success ⊥ physics truth.** MRH's usefulness does *not* depend, in any way, on the
  intent-substrate ontology being correct — you can adopt it believing standard QFT+GR, or the
  Free Energy Principle (which the doc says MRH "aligns with"), or nothing about physics. The
  ontology does zero work *inside* the design pattern. So the applied success provides **no
  evidence** for the physics claims; they are logically independent.
- **Motte vs bailey.** *Bailey* (ambitious): Synchronism is a CFD model of the physical
  universe. *Motte* (defensible, demonstrated): Synchronism's concepts are a useful design
  vocabulary. The motte is true; the bailey is unproven (Bucket 0 = 0); and the motte's truth
  says **nothing** about the bailey, because the vocabulary works regardless of the physics.

This **closes the last shelter** the way the arc closed the physics doors: the "but it delivers"
rejoinder is *true* and does not rescue the physics claim. That stops the oscillation between
"the physics is unproven" and "but look, it's load-bearing" — a productive elimination, not a
gotcha. The framework's robust, demonstrated identity is a **generative design-ontology /
productive-metaphor program**, which is exactly where its effort reliably pays.
