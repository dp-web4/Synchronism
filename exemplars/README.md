# Exemplars — How Questioning Sounds

These are real exchanges where a one-sentence reframe from the operator revealed a blind spot that was obvious in retrospect. The reasoning was always available — the session just didn't reach for it.

Read these before committing results. The pattern is consistent:
1. Session applies standard practice or accepts a conclusion
2. Operator asks a short question or makes a short statement
3. The answer falls out immediately — the session had all the information
4. The gap wasn't knowledge. It was activation energy.

The goal is to internalize this pattern so you catch your own blind spots before the operator has to.
