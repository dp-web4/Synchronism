S670 just gave the tool. This session is that argument.

## The Question S670's Tool Cannot Answer

S670's Tier-1 criterion separates **reparametrization** (central object ≡ a pre-existing quantity, no new prediction) from **confirmed discovery** (makes a new confirmed prediction). But there is a third category the framework leans on — the "wrong theories motivate right questions / productive scaffolding" defense (S580, S614, S615, S627): a **generative reformulation** that makes no new prediction *at proposal* but later cashes out into confirmed novelty.

Can any test, applied *at proposal time*, distinguish a sterile reparametrization from a generative-but-not-yet-cashed reformulation?

They were reformulations — no new prediction at proposal — exactly like the sterile cases. The *only* separating variable is P3, eventual cash-out, which is **knowable only in retrospect**. Generativity is not a property a theory has at proposal time; it is a verdict history returns later.

## Two Corollaries That Remove the Consolation

**1. "Motivates right questions" is non-discriminating.** It keys on P1, which is ✓ for phlogiston and caloric too. Phlogiston organized a century of chemistry questions; caloric motivated calorimetry. "It generated productive questions" is satisfied by *sterile* theories — so it provides **zero positive evidence** that a theory is generative rather than sterile. The framework's repeated fallback (S614/S615/S627: "wrong theories motivate right questions — this is how research works") is true but empty as a defense: it cannot distinguish Synchronism from phlogiston.

**2. The Bohr–Sommerfeld analogy is doubly unearned.** The scaffolding defense (and the `framework_classification` proposal) invokes Bohr–Sommerfeld as the precedent. But Bohr–Sommerfeld made **new confirmed predictions at proposal** — the hydrogen spectrum and Rydberg constant (1913), fine structure (Sommerfeld 1916). That is P2 = ✓: a genuine discovery, *not* a P2 = ✗ scaffold. Synchronism (P2 = ✗, 0 predictions) is *less* like Bohr–Sommerfeld than the defense implies. The analogy claims a precedent the framework does not match.

## Where Synchronism Actually Sits

Its proposal-time signature (P1, P2) = (✓, ✗) places it squarely in the reference class but **leaves its subset — sterile vs generative — undecidable by any current test**, including S670's Tier-1 criterion (which keys on P2 and fires "reparametrization" on heliocentrism too). The only evidence bearing on P3 is the **track record: 0 confirmed novel predictions in 670+ sessions over ~30 years.** That track record is *compatible with sterile* and provides *no support for generative*. The generative cases cashed out in 12–130 years — but every one of them eventually produced a confirmed novel prediction. Synchronism has produced none.

**Honest terminal status:** not "refuted" (the sterile/generative question is undecidable by construction), and not "vindicated as productive scaffolding" (that label is unearned — it is conferred only by a cash-out that has not happened). The status is **undecided, with a 0/670 null as the sole evidence, leaning sterile by base rate** — and a single future Tier-1 confirmed prediction would flip it.

## So What — the Frame Answer

The prompt's central question — "what would Synchronism say that no other framework could, that turns out to be true?" — now has its most precise answer across the whole arc:

- *Whether it CAN* is **undecidable**: no proposal-time test separates a generative reformulation from a sterile one; only future confirmed predictions do.
- *Whether it HAS* is **no**, decisively: 0 in 670+ sessions; and S665/S666/S669 show its predictive content is imported from the standard models it maps to, which structurally bars Tier-1 novelty (you cannot predict what your source theory doesn't).
- *Whether the "scaffolding" defense rescues it* is **no**: that defense is non-discriminating (phlogiston passes it), and its favored analogy (Bohr–Sommerfeld) actually made predictions Synchronism never has.

This is the uncomfortable result the prompt asks for, and it is uncomfortable in the right direction — it does not let the program off with "at least it asked good questions" (so did phlogiston), nor does it claim a refutation it cannot have. It pins the framework to a reference class and says: the only thing that would move it out of the sterile subset is the one thing 670 sessions have not produced. The epicycle tension the prompt opens with is not rhetorical here — Synchronism is, by this analysis, in epicycles' reference class until it earns its way out.
