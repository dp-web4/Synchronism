## Part 2: Mathematical Derivation

### From Phase Field to Wave Function

The intent field at position x and time t has:
- **Amplitude** A(x,t): How much of the pattern exists at x
- **Phase** φ(x,t): Oscillation state at x

Define:
```
ψ(x,t) ≡ A(x,t) × e^{iφ(x,t)}
```

This is a **definition**, not an approximation.

### Extracting Physical Quantities

From ψ, we extract:
- **Amplitude**: A = |ψ|
- **Phase**: φ = arg(ψ)
- **Intensity**: A² = |ψ|²
- **Phase gradient**: ∇φ = Im(∇ψ/ψ)

### The De Broglie Relations Emerge

For a free particle with energy E and momentum p:

**Phase evolution**:
```
φ(x,t) = p·x/ℏ - Et/ℏ
```

This gives:
```
∂φ/∂t = -E/ℏ   (Energy determines phase rate)
∇φ = p/ℏ       (Momentum determines phase gradient)
```

These are the de Broglie relations - **derived, not assumed**.

### The Schrödinger Equation

For a free particle, E = p²/2m:
```
∂φ/∂t = -p²/(2mℏ) = -ℏ(∇φ)²/(2m)
```

With A = constant, this gives:
```
iℏ ∂ψ/∂t = -ℏ²/(2m) ∇²ψ
```
