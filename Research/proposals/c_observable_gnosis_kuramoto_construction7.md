## The Stale-Snapshot Problem

> #6 `C_conv` (Gnosis, belief convergence) — "Independent — input ≠ target… Genuinely tested;

That credit is correct **for the construction it names** — but it is one paradigm out of date.
`C_conv` (inter-agent belief/trust convergence, the S116 trust-entropy / "Generic Corporate
attractor C≈0.45" lineage) is no longer the Gnosis C under active development. Between that work
and June 2026, the Gnosis track migrated — via S73 "Coherence IS Phase" (the same-day answer to

tunable coupling `κ_C` and matched-filter detection of an injected coherence signal. S116–S119 are
that program. S690 audited the retired C and did not see the live one.

## What the Live Gnosis C Actually Is (and isn't)

The S116–S119 apparatus is a **Monte-Carlo simulation of a coupled phase-oscillator array**
(32 seeds × 25 chunks × 1.01M steps/seed; κ_p=0.30, ω_std=0.07 fixed; injected signal amplitude
`A`; matched-filter detection scored against paired `A=0` nulls). Two consequences for S690's
taxonomy:

1. **It is a forward map, not an independent measurement.** The "signal" detected is the signal
   *injected*. Input is not independent of target — it fails the very input≠target criterion that
   earned `C_conv` its row-#6 credit. This is a detectability characterization of a *tuned model
   parameter*, not a calibration of C against ground truth.

2. **Its own conclusion is that the quantity is not a clean observable** — reached independently of
   S690, in this framework's adjacent dialect:
   - **S117**: the headline ρ_sig "collapse" is an *effect-size illusion* — "detection never
     failed." The apparent observable was an artifact of coherent class-combination, not a property
     of C.
   - **S118**: "the detection edge is **detector-relative** — no intrinsic over-coupled collapse."
     The measured value depends on which detector is chosen.
   - **S119**: d′ ≈ √2 / CV_sig — the effect size is set *entirely by the signal class's own
     seed-to-seed coefficient of variation* (a "reproducibility limit"), while AUC stays pinned at
     1.0 across the whole range.
